### Atomic Structure

**Key Concepts:**
- **Atomic Number (Z)**: Number of protons (defines the element)
- **Mass Number (A)**: Protons + Neutrons
- **Isotopes**: Same protons, different neutrons
- **Ions**: Atoms that gain or lose electrons
  - Cations: Positive (lose electrons)
  - Anions: Negative (gain electrons)

**Important Formulas:**
- Neutrons = Mass Number - Atomic Number
- \( \text{Neutrons} = A - Z \)

**Atomic Notation:**
- \( ^A_Z X \) where X is the element symbol

**Electron Configuration Rules:**
1. Aufbau Principle: Fill lowest energy orbitals first
2. Pauli Exclusion Principle: Max 2 electrons per orbital
3. Hund's Rule: Fill orbitals singly before pairing

### Chemical Equilibrium

**Equilibrium Constant (K):**
- \( K_c = \frac{[\text{products}]}{[\text{reactants}]} \) (with coefficients as exponents)
- For \( aA + bB \rightleftharpoons cC + dD \):
  - \( K_c = \frac{[C]^c[D]^d}{[A]^a[B]^b} \)
- **Exclude**: Pure solids and pure liquids from K expression

**Reaction Quotient (Q):**
- Same formula as K, but uses current concentrations
- **Q < K**: Forward reaction favored (shift right)
- **Q = K**: System at equilibrium
- **Q > K**: Reverse reaction favored (shift left)

**Le Chatelier's Principle:**
When stress is applied to equilibrium, system shifts to relieve stress.

| Stress | Equilibrium Shifts |
|--------|-------------------|
| Add reactant | â†’ Products (right) |
| Add product | â† Reactants (left) |
| Remove reactant | â† Reactants (left) |
| Remove product | â†’ Products (right) |
| Increase pressure | â†’ Fewer gas moles |
| Increase temperature | Endo: â†’, Exo: â† |
| Add catalyst | No shift (reaches equilibrium faster) |
